Swift Playgrounds 4 release imminent

Finally seems like the wait is about to get over and soon you will be able to install Swift Playgrounds 4 on your iPads. Apple.com still mentions vague “available late 2021” release date, but you should expect the release within a couple of weeks. Why? Here are some points to back it:

  1. Xcode 13.2 beta (released initially on October 27th) has support for “app projects created with Swift Playgrounds 4”.
  2. As reported by 9to5mac, Apple has invited some developers to test Swift Playgrounds 4 beta via TestFlight. Also the article mentions “some of the app’s features require iPadOS 15.2”, and that leads to the third point.
  3. iPadOS 15.2 beta 4 was released last week, and from the release schedule of recent betas we should expect the next release (most probably Release Candidate or final) within two weeks.

Although everything is pointing to an imminent release, still there could be some timeline changes if Apple is not happy with the feedback from beta testers.
